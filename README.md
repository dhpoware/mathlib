# mathlib
The main files that make up this 3D math library are:
- mathlib.h
- mathlib.cpp
- collision.h
- collision.cpp

All other files are part of the testing framework used to test the library.

The core math classes include:
- Math
- Vector2
- Vector3
- Vector4
- Matrix3
- Matrix4
- Quaternion
- MatrixStack

The collision classes include:
- BoundingBox
- BoundingSphere
- BoundingVolume
- Plane
- Frustum
- Ray
